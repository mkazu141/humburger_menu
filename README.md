# humburger_menu
そのまま使えるハンバーガーメニュー
index.html内のsectionに要素を配置して使用する
```
            <section>
                <div id="top">
                </div>
                <div id="article">
                </div>
                <div id="button">
                </div>
            </section>
```
デフォルトでは三つの要素にページ内リンク移動するように設定している
```
            <nav class="sp_menu">
                <ul>
                    <a href="#top"><li>TOP</li></a>
                    <a href="#article"><li>使い方</li></a>
                    <a href="#button"><li>利用開始</li></a>
                </ul>
            </nav>
```
